# Proyecto Final React JS Bastian Bastias - Redux

Sistema realizado con React JS, utilizando la librería Redux para el manejo de estados de la aplicación.

El sistema actualmente lista todas las ofertas de Steam que nos brinda la api [CheapShark](https://apidocs.cheapshark.com/), ordenando la información desde el mayor descuento al menor.
Podremos visualizar el detalle de cada Deal y guardar todas las ofertas que nos interesen.
Las ofertas listadas son almacenadas tanto en el localstorage como en un estado general apoyado por Redux.
Cada oferta posee un estado interno de favoritos el cual nos permitirá luego poder identificar los juegos que deseamos.

## Otros

El sistema cuenta con una validación de ruta 'NotFound' en caso de ingresar a una URL no valida, redireccionará automaticamente a esta Ruta.
Cuenta con un lazyloading sobre nuestro componente de Deseados con la finalidad de optimizar la carga principal del App.
Tambien tiene implementado un limitador de errores : HOC ErrorBoundary en nuestro listado de Deals, en caso de fallar o haber un error, sera manejado y mostrara nuestra interfaz definida para ello.

Arquitectura basada en patrones, aplicando buenas prácticas de Code Splitting.
